A few years ago, I published [a list of good options for sharing code snippets](https://terminalroot.com/best-alternatives-for-sharing-code-snippets/).

Now, with new tools emerging and others evolving, I've prepared an updated version: **20 Alternatives for Sharing Code as Images**. If you've seen the old list, this is a continuation, but with IMAGES. If you haven't, this new selection already includes what's worth using today.

## 01. [Ray.so](https://ray.so/)
Very clean and simple interface. Has light, dark, and gradient backgrounds. Allows adjusting padding, font size, and exporting as PNG.

![01](/assets/img/dicas/code-img/01.jpg)

---

## 02. [CodeImage (app.codeimage.dev)](https://app.codeimage.dev/)
Highly configurable: themes, borders, windows, fonts. Great for those who want to "build" the image, not just copy and paste.

![02](/assets/img/dicas/code-img/02.jpg)

---

## 03. [Snappify](https://snappify.com/)
Allows for more complex layouts: multiple code blocks, annotations, diagrams. Exports in PNG or SVG.

![03](/assets/img/dicas/code-img/03.jpg)

---

## 04. [CodetoImg](https://codeimg.io/)
Simple ("no-frills") interface for quickly turning code into an image. Allows for different themes and languages.

![04](/assets/img/dicas/code-img/04.jpg)

---

## 05. [Chalk.ist](https://chalk.ist/)
Has interesting modes like "diff mode" (showing differences between code versions). Allows customization of background, line numbers, reflection, etc.

![05](/assets/img/dicas/code-img/05.jpg)

---

## 06. [CodePNG](https://www.codepng.app/)
Minimalist tool for generating simple and elegant images. Support for different languages, customizable font, and simple settings.

![06](/assets/img/dicas/code-img/06.jpg)

---

## 07. [Polacode (VS Code)](https://marketplace.visualstudio.com/items?itemName=pnp.polacode)
Extension for VS Code: generates a screenshot directly from the editor, preserving your theme. Very useful if you want to capture code snippets while you're already programming.

![07](/assets/img/dicas/code-img/07.jpg)

---

## 08. [Germanium](https://github.com/matsuyoshi30/germanium)
A CLI tool for generating code images.

![08](/assets/img/dicas/code-img/08.jpg)

---

## 09. [CodeKeep](https://codekeep.io/screenshot)
Generates organized images of snippets with a professional appearance.

![09](/assets/img/dicas/code-img/09.jpg)

---

## 10. [CodeSnap](https://codesnap.dev/)
Captures code snippets with highlighting and styled borders.

![10](/assets/img/dicas/code-img/10.jpg)

## 11. [ShowCode](https://showcode.app/)
Transforms code into customized images for sharing.

![11](/assets/img/dicas/code-img/11.jpg)

---

## 12. [Pika Code](https://pika.style/templates/code-image)
Offers modern templates for generating code images.

![12](/assets/img/dicas/code-img/12.jpg)

---

## 13. [Code to Image Converter](https://10015.io/tools/code-to-image-converter)
Quick tool for converting code into adjustable images.

![13](/assets/img/dicas/code-img/13.jpg)

---

## 14. [HackReels](https://www.hackreels.com/)
Creates short, dynamic code videos for social media.

![14](/assets/img/dicas/code-img/14.jpg)

---

## 15. [Codebit](https://codebit.xyz/)
Allows saving and sharing snippets in a clean visual style.

![15](/assets/img/dicas/code-img/15.jpg)

---

## 16. [SnippetsLab](https://www.renfei.org/snippets-lab/)
Complete snippet manager for macOS.

![16](/assets/img/dicas/code-img/16.jpg)

---

## 17. [Marmoset](https://chromewebstore.google.com/detail/marmoset/npkfpddkpefnmkflhhligbkofhnafieb?pli=1)
Extension that generates styled screenshots directly in the browser.

![17](/assets/img/dicas/code-img/17.jpg)

---

## 18. [SnippetShare](https://snippetshare.dev/)
Simple and fast sharing of code snippets.

![18](/assets/img/dicas/code-img/18.jpg)

---

## 19. [Petrify](https://www.dangercove.com/petrify/)
Generates elegant code images with configurable themes.

![19](/assets/img/dicas/code-img/19.jpg)

---

## 20. [Source Code Shots](https://sourcecodeshots.com/)
Creates professional snippet images for posts and portfolios.
